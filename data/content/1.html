Цикл for

Обратите внимание на синтаксис — условия разделяются точкой с запятой, но скобки для тела цикла можно не ставить, если действие одно [citation:5].

// Создаем 10 врагов
for (var i = 0; i < 10; i += 1) 
{
    instance_create_layer(100 + i*50, 100, 'Instances', obj_enemy);
}

Условия if

Забавная особенность GML — можно использовать и один, и два знака равенства [citation:5].

// Один знак '=' тоже работает для сравнения!
if (health = 0) 
{
    show_debug_message('Игрок мертв');
    instance_destroy();
}
